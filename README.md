# TravelApp
## Description
This travel application allows users to book their desired trip, see the weather at their desired destination, add a packing list, and set a budget 
## Technologies used
- HTML
- CSS
- JS (ES6)
- Jquery
## Required Dependencies 
- Jquery 3.5
- apis used :
    - unsplash
    - google maps 
    - openweather api
